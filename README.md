UnityTools is an ongoing side project for utilities I create while working on my multiplayer fps towerdefense game.

You can now put these scripts in your unity using package manager's `Install package from Git URL...`

`https://github.com/s1ckboii/UnityTools.git#dev`

I will have some markdowns in each folder with pictures, detailed info on what they are, why are they good, etc.

Markdowns in `Assets\` will describe what scripts you'll find in each folder that has the same name in Editor or Runtime.

You CAN use these scripts on their own, but I highly suggest to expand on them so you'll have something unique, and also more inline with what your game needs.

All of these code is fully FREE to use, even the small icons.

If you have any suggestions what to add, let me know and I'll think about it. You are also free to PR new additions, or improved versions of existing components.
